## 📖 Обзор

**EventHub** — это современная высокотехнологичная платформа для организации видеовстреч. Фронтенд приложения реализован на Angular 21+ с использованием **Signals** для максимально производительного и реактивного управления состоянием. Интерфейс выполнен в уникальном стиле **Neobrutalism**, что выделяет проект среди стандартных корпоративных решений.
Бэкенд на **Java 21** обеспечивает высокую пропускную способность, предоставляя богатый REST API и WebSocket-эндпоинты для управления комнатами, пользователями и обмена сообщениями в реальном времени. Интеграция с **LiveKit** и **WebRTC** гарантирует стабильную потоковую передачу аудио и видео с минимальной задержкой.
Система идеально подходит для бизнес-встреч, удаленного обучения и неформального общения в современном цифровом пространстве.

## 🛠 Технологический Стек

### Frontend
- **Angular 21+** — Фронтенд-платформа с использованием Signals для эффективного управления состоянием
- **TypeScript** — Строгая типизация на стороне клиента
- **RxJS** — Реактивная обработка асинхронных потоков данных на фронтенде
- **LiveKit** / **WebRTC** — Передача видео и аудио с ультранизкой задержкой
- **WebSocket** — Мгновенный обмен сообщениями и синхронизация состояния комнат

### Backend
- **Spring Boot 4** — Высокопроизводительный бэкенд на современном стеке
- **Java 21** — Язык программирования
- **PostgreSQL** — Основное хранилище данных (пользователи, комнаты, история)
- **Redis** — Кэширование сессий и Pub/Sub для масштабирования веб-сокетов
- **WebSocket** — Протокол передачи данныех в реальном времени
- **JWT** (JSON Web Tokens) — Защищенная Stateless-аутентификация
- **Docker** & **Docker Compose** — Контейнеризация всех сервисов для быстрого развертывания одной командой
- **Gradle** — Управление сборкой и зависимостями бэкенда

## ✨ Ключевые Возможности

### 🏠 **Управление Видеокомнатами**
- **Создание комнат "в один клик"** через интуитивно понятный API
- **Генерация уникальных ссылок-приглашений** для безопасного и простого доступа
- **Гибкие настройки комнаты:** заголовок, описание, лимит участников, публичный/приватный доступ
- **Гостевой доступ** — присоединиться к встрече можно без регистрации
- **Валидация и проверка** существования комнат для обеспечения безопасности

### 🔐 **Система Аутентификации и Безопасности**
- **JWT-аутентификация** с access/refresh токенами
- **Полный цикл регистрации и входа**
- **Восстановление и сброс пароля** через email
- **Ролевая модель:** Администратор, Пользователь, Гость

### 👥 **Управление Пользователями и Участниками**
- **Полноценные профили пользователей**
- **Гостевые сессии** с временными профилями
- **Динамический список участников:** Отслеживание всех пользователей в комнате в реальном времени
- **Детальная информация об участниках:** статус, права, состояние аудио/видео

### 🎥 **Высококачественные Медиапотоки (LiveKit & WebRTC)**
- **Передовая технология WebRTC** для передачи аудио и видео с минимальной задержкой
- **Сигнальный сервер** на WebSocket для установления P2P-соединений
- **Стабильность соединения:** Надежная обработка событий подключения и отключения

### 💬 **Текстовый Чат в Реальном Времени**
- **Архитектура SFU (Selective Forwarding Unit):** Использование LiveKit для стабильной передачи аудио и видео многим участникам одновременно.
- **Полная история чата:** Все сообщения сохраняются на время сессии
- **Умные уведомления** о новых сообщениях
